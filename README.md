# ◈ Nimbus — Weather Intelligence Dashboard

A sleek, modern **weather intelligence dashboard** that delivers real-time weather insights for any city on Earth.
Nimbus combines **minimalist design, smooth UI interactions, and powerful weather APIs** to create a fast and elegant weather experience.

Built with **vanilla JavaScript, modern CSS, and the OpenWeatherMap API**, Nimbus focuses on performance, clarity, and usability.

---

## ✨ Features

### 🌍 Global Weather Search

Search for **any city worldwide** and instantly view current weather conditions.

### 📊 Real-Time Weather Data

Powered by the **OpenWeatherMap API** to deliver accurate and up-to-date weather data.

### 📅 5-Day Forecast

See a clean forecast overview for the upcoming days.

### 📍 Geolocation Weather

Automatically detect the user's location and show local weather conditions.

### 🌗 Dark / Light Mode

Toggle between elegant **dark and light themes** with smooth transitions.

### 🌡 Temperature Unit Switch

Instantly switch between **Celsius and Fahrenheit**.

### 🔍 Smart Search History

Nimbus remembers recent searches and lets users quickly revisit them.

### 📱 Fully Responsive Design

Optimized for **desktop, tablet, and mobile devices**.

### ⚡ Fast & Lightweight

No frameworks. Just **clean, optimized vanilla JavaScript**.
